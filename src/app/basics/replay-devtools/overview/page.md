## Overview

When you open a replay, you're able to inspect your application with Replay DevTools as if you were there when it initially ran because there is a browser running in the cloud that thinks it's running at the original point in time on the original computer. [Learn more about how this works.](/basics/overview/how-does-time-travel-work)
